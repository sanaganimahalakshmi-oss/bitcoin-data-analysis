# bitcoin-data-analysis
# Bitcoin Market Sentiment Analysis

## Objective
This project analyzes the relationship between Bitcoin market sentiment (Fear & Greed Index) and trader performance using Hyperliquid historical trading data.

## Datasets
- Historical Hyperliquid Trading Data
- Bitcoin Fear & Greed Index

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook (VS Code)

## Project Workflow
1. Import libraries
2. Load datasets
3. Explore the data
4. Clean the data
5. Convert timestamps
6. Merge datasets
7. Perform Exploratory Data Analysis (EDA)
8. Create visualizations
9. Generate business insights

## Key Insights
- Fear sentiment had the highest number of trades.
- Trading profitability varied across different market sentiments.
- Market sentiment provides useful information for analyzing trading behavior.
- Some recent trades could not be matched because the sentiment dataset ended before the trading dataset.
